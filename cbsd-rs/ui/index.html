Browser Access Restricted

This endpoint is designed for programmatic access only.
Please download the CLI tool to interact with this server.

⬇ Download CLI (Linux) 🍎 Download CLI (macOS ARM64)
  • ▸
    Download the appropriate CLI tool for your platform.
  • ▸
    cp cbc-{platform} cbc && chmod +x cbc
  • ▸
    Run with cbc --help

Quick Start

$ cbc login

What happens next?

  • ▸
    SSO Redirect: The tool provides a URL. Open it in your browser to sign in with SSO using your credentials.
  • ▸
    Auto-Configuration Token: Upon successful login, a token will be presented in the browser. Copy this token and paste it back into the CLI when prompted.
  • ▸
    Validation: Run cbc whoami to ensure your login is working.
  • ▸
    Configuration File: The configuration file should be present in the user's home configuration directory, in cbc/config.json.
