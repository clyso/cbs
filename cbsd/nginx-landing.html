CLYSO

Browser Access Restricted

This endpoint is designed for programmatic access only.
Please download the CLI tool to interact with this server.

⬇ Download CLI (Linux)
Simply download, chmod +x cbc, and run.

Quick Start

$ cbc auth login https://cbs.ces.clyso.cloud

What happens next?

  • ▸
    SSO Redirect: The tool provides a URL. Open it in your browser to sign in via Google SSO using your Clyso credentials.
  • ▸
    Auto-Configuration: Upon successful login, a cbc-config.yaml file will automatically download to your computer.
  • ▸
    Placement: Ensure cbc-config.yaml is in the same folder as the tool, or specify its path using the -c flag (e.g., cbc -c /path/to/config ...).
